## Requirements

- Python 3.x
- Pip 3.x

```bash
pip3 install -r requirements.txt
```

## Run

```bash
python3 main.py path/to/impulse/responses/dir path/to/dry/signals/dir [path/to/output/dir]
```

or as a script

```bash
chmod 744 main.py
./main.py path/to/impulse/responses/dir path/to/dry/signals/dir [path/to/output/dir]
```
